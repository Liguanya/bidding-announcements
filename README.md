# 招标公告推送平台

> 实时推送政府采购招标公告，助您把握商机

## 📋 功能特性

- ✅ **公告列表展示** - 展示每日搜索到的招标公告
- ✅ **多维度筛选** - 按日期、关键词、来源网站筛选
- ✅ **全文搜索** - 支持搜索标题、单位、关键词等
- ✅ **公告详情** - 点击查看完整公告信息
- ✅ **一键复制链接** - 快速复制公告链接
- ✅ **导出Excel** - 将筛选结果导出为CSV文件
- ✅ **收藏功能** - 本地收藏重要公告（localStorage）
- ✅ **响应式设计** - 支持手机和电脑访问
- ✅ **无需登录** - 完全公开，公司同事可直接访问

## ⏰ 自动化更新

### 安装定时任务

```bash
cd 招标公告推送平台/scripts
bash install_cron.sh
```

### 定时任务说明

- **执行时间**：每天 09:00、14:00、18:00
- **执行内容**：读取 `造价咨询/招标公告监控/` 目录下的数据，生成 `data/announcements.json`
- **日志文件**：`logs/update.log`

### 常用命令

```bash
# 查看定时任务
crontab -l

# 编辑定时任务
crontab -e

# 删除定时任务
crontab -r

# 手动执行更新
python3 scripts/update_data.py

# 查看日志
tail -f logs/update.log
```
